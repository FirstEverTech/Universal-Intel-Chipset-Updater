# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in this project, please report it privately:
- Email: contact@firstever.tech
- Do NOT create public GitHub issues for security vulnerabilities

## Security Measures

- All downloaded files are verified against SHA-256 hashes
- System restore points are created before installation
- Extensive logging for audit purposes
- Requires administrator privileges for system modifications

## Supported Versions

Only the latest version receives security updates.
